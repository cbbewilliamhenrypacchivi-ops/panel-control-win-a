🔧 Características Adaptables

Características adaptables observadas en https://portfolywin-a.netlify.app/:

✓ Menú responsive (≤ 900px)
En pantallas de 900px o menos se ocultan los enlaces del menú y se muestra un botón de despliegue (toggle) para navegación móvil.
✓ Reflujo de tarjetas (≤ 720px)
El layout de tarjetas tipo “job-card” cambia a una sola columna para mejorar lectura y scroll vertical en teléfonos.
✓ Contenedores fluidos
Se usan contenedores con anchos flexibles (por ejemplo, patrones tipo width: min(..., 100%)) para evitar desbordes y mantener márgenes consistentes.
✓ Tipografía fluida
Se emplea tipografía responsiva (por ejemplo con clamp()) para mantener jerarquía y legibilidad entre escritorio y móvil.
✓ Preferencias de accesibilidad
Existe soporte para prefers-reduced-motion, reduciendo animaciones y transiciones cuando el usuario lo solicita.

🧪 Pruebas de Compatibilidad

Se ha testeado la compatibilidad en navegadores principales: Chrome, Firefox, Safari, Edge. Además, se verifica en dispositivos reales y emuladores para garantizar la mejor experiencia posible en todas las plataformas.
