🎯 Principios de Diseño

Claridad: La interfaz utiliza contraste alto para legibilidad óptima.

Consistencia: Los elementos visuales se repiten de manera uniforme en toda la plataforma.

Modernidad: Animaciones suaves y transiciones crean una experiencia fluida.

Accesibilidad: Se cumplen estándares WCAG para usuarios con diferentes capacidades.
